About Me

My Brief Introduction

Hi, I’m Yashwanth K, a BCA Graduate and Junior Software Engineer from Bangalore who genuinely enjoys building things on the web. I’m passionate about Full Stack Development because it allows me to work on both how an application looks and how it works behind the scenes. I like creating websites that feel simple, smooth, and easy for users to interact with. On the frontend, I work with React to build responsive and interactive user interfaces. On the backend, I use Node.js to handle server-side logic and APIs. I also work with PostgreSQL for managing data and designing structured databases. Through these tools, I build applications that are practical, functional, and ready for real use. Along with web development, I’ve started working with cloud technologies, mainly Google Cloud Platform (GCP). I’ve had hands-on exposure to deploying applications and exploring basic GCP services as part of my learning and work. This has helped me understand how applications run beyond just local environments. Working in a corporate setup has taught me how real development teams function. I’ve learned the importance of clean code, communication, and asking the right questions. I’m always eager to learn, improve, and take feedback to grow as a developer. I enjoy building things that solve real problems and make life a little easier—and more fun. My goal is to keep learning, improve my skills, and grow into a strong full stack engineer with solid cloud knowledge.
